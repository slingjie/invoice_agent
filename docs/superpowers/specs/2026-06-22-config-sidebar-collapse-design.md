# 任务配置侧栏收起设计

## 目标

在桌面端为左侧“任务配置”面板增加收起和展开按钮。收起时压缩左栏，为右侧执行状态与确认预览区域释放宽度。

## 交互

- 按钮位于“任务配置”标题区域右侧。
- 默认保持展开，不改变现有页面初始布局。
- 点击后切换 `aria-expanded`、按钮说明和页面收起状态。
- 收起状态仅保留窄侧栏和展开按钮，隐藏表单内容。
- 视口宽度不超过 860px 时保持单列布局，不启用侧栏收起。

## 实现边界

- 使用现有 HTML、CSS 和原生 JavaScript，不增加依赖。
- 不增加拖拽调宽。
- 不持久化收起状态。
- 不修改任务提交、扫描、预览或导出逻辑。

## 验收

- 页面包含可访问的侧栏切换按钮。
- 桌面端点击可在展开和收起状态间切换。
- 收起后右侧区域获得更多可用宽度。
- 移动端继续使用现有单列布局。
- 现有自动测试通过，并在 `http://127.0.0.1:8765/` 完成真实入口验证。
