# PRD 已批准决策补充

本文件补充 `prd.md` 中访谈后确认的最终产品规则。

- MVP 只导出已有有效票据和自动餐补，不增加无票现金项目。
- 输出 `01_公司报销单.xlsx`。
- Windows 且安装 Microsoft Excel 时同步输出合并的 `01_公司报销单.pdf`。
- PDF 不可用时跳过并提示；转换失败时保留两份 Excel，导出结果为部分成功。
- 使用内置清理后的正式模板，不修改用户参考文件。
- 报销明细表为 A4 纵向；差旅费和日常费用报销单为 A5 横向。
- 差旅交通每页 12 条，日常费用每页 8 条；无日常费用时不生成空白页。
- 报销明细分类空间不足时动态插行并更新公式和打印区域。
- 填报日期为导出当天固定值，金额大写由程序生成固定中文文本。
- 签字栏仅填写报销人，其余审批栏保持空白。
- 基础信息缺失或疑似占位值不阻断导出，只提示用户人工修改。
- 有效票据即使超出出差日期仍导出。
- 关联行程单不计金额和张数，只作为附件保留。
- 公司 Excel 或 PDF 失败时保留之前成功生成的文件，并明确显示部分成功及原因。
